# Fletcher Lawrence - Academic Portfolio

This repository contains the source code for the academic and professional portfolio of Fletcher Lawrence, PhD Student at the University of Florida.

## Contents
* **index.html** - The main HTML file containing the structure, content, and styling.
* **WEBP & PNG Images** - Compressed, web-friendly images for the project cards and gallery.
* **PDFs** - Research publications and CV linked directly on the site.

## Local Development
Since this is a static site without an active build process or external dependencies, you can run it locally by either:
1. Using the provided `server.ps1` PowerShell script to launch a local server.
2. Using any simple HTTP server (e.g. `python -m http.server`, `npx serve`, or the VS Code Live Server extension).

## Export to GitHub Pages
To make this site publicly accessible:
1. Create a new, empty repository on GitHub.
2. Push this existing code to your new remote repository using the instructions provided by GitHub.
3. In your GitHub repository's settings, navigate to the **Pages** section and select the `main` branch as your source to deploy the website.
